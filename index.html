Projects

Smart Blind Stick Using UNO NANO

The Smart Blind Stick using Arduino Nano is an assistive device designed to help visually impaired individuals navigate safely. It utilizes ultrasonic sensors to detect obstacles and provides feedback through a buzzer or vibration motor. Additional features like water detection and GPS tracking can be integrated for enhanced functionality. This compact and cost-effective solution improves mobility and independence for the visually impaired.

Microstrip Patch Antenna for X-Band VSAT Applications

The Microstrip Patch Antenna for X-Band VSAT applications is a compact, lightweight, and high-gain antenna designed for satellite communication. It operates in the X-band frequency range, ensuring reliable signal transmission and reception. Its low-profile design makes it suitable for modern VSAT systems. The antenna offers efficient performance with minimal power loss.
